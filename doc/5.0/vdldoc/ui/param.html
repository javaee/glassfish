ui

Tag param

  • Description:

    Use this tag to pass parameters to an included file (using ui:include), or a template (linked to either a composition or decorator). Embed ui:param tags in either ui:include, ui:composition, or ui:decorate to pass the parameters.

Attributes 
NameRequiredTypeDescription
nametruejavax.el.ValueExpression
(must evaluate to java.lang.String)
The name of the parameter.
valuetruejavax.el.ValueExpression
(must evaluate to java.lang.String)
The value of the parameter. Notice that this attribute's value can be an EL expression, which means that you can pass objects to either an included file or a template.
