Sun GlassFish Enterprise Server - Installation Complete

Use the Administration Console to register Sun GlassFish™ Enterprise Server (GlassFish server) with Sun SM Connection now: As a registered user, you receive benefits such as:

  • Patch information and bug updates
  • Screencasts and tutorials
  • News and events
  • Support and training offerings

Quick Start Guide: Start here to learn about the GlassFish server. The Quick Start Guide explains how to perform basic tasks such as starting the server, accessing the Administration Console, and deploying a sample application.

Update Tool: Use the Update Tool to install and update additional technologies and frameworks such as:

  • Metro
  • Jersey
  • jMaki
  • JRuby runtime

The Update Tool is developed through the Update Center project.

To improve the user experience and optimize offerings to users, Sun collects data about GlassFish server usage that is transmitted by the Update Tool installer as part of the automatic update processes. No personally identifiable information is collected by this process.

Product documentation: View the online product documentation for complete information about the GlassFish server, including up-to-date release notes.

Java™ Platform, Enterprise Edition (Java EE) API online documentation: View the API reference documentation for the GlassFish server packages and related APIs. This documentation is generated from the source code by the Javadoc™ tool.

The Java EE Tutorial: A beginner's guide to developing enterprise applications for the GlassFish server. It includes working examples and step-by-step instructions for creating Java EE applications with these technologies: Web services, Java Servlets, JavaServer Pages (JSP), JSP Standard Tag Library (JSTL), Enterprise JavaBeans, Java Message Service, and Java EE Connector Architecture.

Java BluePrints: Provides practical design advice on enterprise and wireless Java applications, for developers who have mastered basic Java EE concepts.

Features and resources: Learn more about key features of the GlassFish server, including what's new; plus additional resources such as forums, mailing lists, support, and how to submit feedback.
