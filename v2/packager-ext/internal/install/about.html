Sun ONE Application Server 7
 
Sun[tm] ONE
The Sun[tm] Open Net Environment (Sun ONE) is Sun's standards-based software vision, architecture, platform, and expertise for building and deploying Services on Demand. Sun ONE Application Server 7 is the new, Java[tm] 2 Platform, Enterprise Edition (J2EE) 1.3 compliant application server that combines key Sun sourced technologies to provide a developer friendly yet extremely performant web services platform.
 
  What's New in Sun ONE Application Server 7
Latest Java Specification Support
Sun ONE Application Server 7 supports the latest J2EE 1.3 specifications and leverages performance enhancements and features of the very latest release of the Java 2 Platform, Standard Edition (J2SE) 1.4 including API enhancements such as the new logging API.
 
Integrated Web Services Support
Standard Java-based web services support is built into Sun ONE Application Server 7. Out of the box, developers can take advantage of a deployment ready web services platform based on the new Java APIs for XML-based RPC (JAX-RPC) and XML Messaging (JAXM).
 
Integrated HTTP and Messaging Services
The high performance HTTP engine from the Sun ONE Web Server has been integrated into the same process as the J2EE web and EJB containers. This integration yields both a streamlined runtime configuration and performance gains for demanding web services and web applications.

The proven Sun ONE Message Queue technology has been integrated into Sun ONE Application Server as the built-in Java Message Service (JMS). Applications built using Message Driven Beans and the JMS API are able to leverage the proven infrastructure of this commercial technology.
 
Enhanced Developer Features
Sun ONE Application Server 7 includes a variety of enhancements for developers:
Updated Sun ONE Studio Integration: An updated version of the Forte for Java Plugin supports J2EE 1.3 development in conjunction with Sun ONE Application Server and the Sun ONE Studio 4 IDE.
Hot Deployment and Redeployment: To speed up development cycle times, dynamic updates can now be made to application files during development without requiring a restart of the application server.
New XML-based Server Configuration: Developers can take advantage of the new file-based configuration approach to easily understand server settings and to quickly make configuration changes.
New JavaServer Pages[tm] and Servlet Page Caching: Sun ONE Application Server 7 makes it easy to speed up web applications by caching results of JSPs and servlets.
New Lifecycle Listener Classes: Going beyond simple startup and shutdown classes, Lifecycle Listener Classes enable developers to extend the services provided by Sun ONE Application Server 7 at multiple points in the application server's lifecycle.
New Ant Tasks: Tasks specific to Sun ONE Application Server 7 deployment and administrative functions can be used to support scripted build environments.
 
Streamlined Installation
Installation of Sun ONE Application Server 7 has been dramatically simplified by no longer requiring a pre-installed web server. A directory server is no longer required for managing server configuration, but can still be used for user authentication.
 
Improved Administration
New Administrative User Interfaces: Sun ONE Application Server 7 includes a new web-based administrative console that exposes all of the underlying server configuration settings. A companion, remoteable command line interface introduces scripting support and remote monitoring capabilities.
Pluggable Authentication: Administrators can now plug in either one of several pre-built authentication modules or use custom modules to carry out user authentication.
Enhanced Log Messages: Each message contains a unique log message ID that can be used to filter events of interest and to quickly look up the cause of error conditions.
