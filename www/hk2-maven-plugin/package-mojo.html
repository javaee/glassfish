hk2:package

Full name:

com.sun.enterprise:hk2-maven-plugin:0.3.37-SNAPSHOT:package

Description:

Creates a jar with a special manifest entry.

Mostly copied from JarMojo. See http://jira.codehaus.org/browse/MNG-2789 why we can't rely on it.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
classesDirectory File - Directory containing the classes.
finalName String - Name of the generated JAR.
outputDirectory File - Directory containing the generated JAR.

Optional Parameters

Name Type Since Description
archive MavenArchiveConfiguration - The maven archive configuration to use.

See the Javadocs for MavenArchiveConfiguration .
excludes String[] - Ant glob patterns to be excluded from the jar file, like "**/*.bak" The base directory for the pattern is specified in classesDirectory.
generateOSGiHeaders boolean - By default, we don't generate OSGi manifest Default value is: false.
resolution String - Whether to generate optional dependency or mandatory dependency Default value is: mandatory.
visibility String - Whether to reexport or not Default value is: private.

Parameter Details
