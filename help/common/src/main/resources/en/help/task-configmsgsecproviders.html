To Configure Enterprise Server Facilities for Use by Message Security Providers

  1. If using a username token, configure a user database, if necessary.

    When using a username/password token, first configure an appropriate realm and an appropriate user database for the realm.

    Configuring a user database is discussed in To Edit a Realm.

  2. Manage certificates and private keys, if necessary.
Next Steps

Once the facilities of the Enterprise Server are configured for use by message security providers, the providers installed with the Enterprise Server may be enabled as described in To Edit a Message Security Configuration.
