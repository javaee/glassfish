To Delete a Lifecycle Module

  1. In the navigation tree, select the Lifecycle Modules node.

    The Lifecycle Modules page opens.

  2. On the Lifecycle Modules page, select the checkbox to the left of the lifecycle module that you are deleting.

  3. Click Delete.
