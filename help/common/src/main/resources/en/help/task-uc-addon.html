To Install Add-On Components

Installing an add-on component involves the following operations:

  • Transferring the content of the component and the component's dependencies from a network repository

  • Installing the component under the GlassFish Server installation image on your system

If a component requires any other components that are not already installed, the required components are also installed.

  1. In the navigation tree, select the Update Tool node.

    The Available Add-Ons page opens.

  2. On the Available Add-Ons page, in the table of available add-on components, select the components that you are installing.

    If no add-on components are available, the table is empty.

    • To select an individual component, select the checkbox adjacent to the name of the component.

    • To select all components, click the Select All icon in the table header.

    • To deselect all components, click the Deselect All icon in the table header.

  3. Click Install.

    A dialog box asks you to confirm that the selected components and any components that they require should be installed.

  4. In the dialog box, click OK.

    You are asked to accept the license agreements of the components that you are installing.

  5. Accept the license agreements of the components that you are installing.

    GlassFish Server confirms that the installation is complete. The components are removed from the table of add-on components and are added to the table of installed components on the Installed Components page.

  6. Restart the server.

    For detailed instructions, see the following documentation:

    • To Stop a Domain in Oracle GlassFish Server Administration Guide

    • To Start a Domain in Oracle GlassFish Server Administration Guide
