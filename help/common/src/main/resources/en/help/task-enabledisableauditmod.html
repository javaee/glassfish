To Enable or Disable Audit Logging

  1. In the navigation tree, expand the Configuration node.

  2. Under the Configuration node, select the Security node.

    The Security page opens.

  3. On the Security page, select the Audit Logging Enabled checkbox to enable audit logging.

    To activate audit logging, you must also set the auditOn property for the active audit module to true.

  4. Click Save.
