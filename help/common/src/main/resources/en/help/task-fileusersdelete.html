To Delete a User from the File Realm

  1. Under the Configurations node, expand the server instance or cluster configuration for which you want to manage Admin Realm users.
  2. Expand the Security node.
  3. Expand the Realms node.
  4. Select the file node.

    The Edit Realm page opens.

  5. On the Edit Realm page, click the Manage Users button.

    The File Users page opens.

  6. On the File Users page, select the checkbox to the left of the name of the user to be deleted.
  7. Click Delete.

    A dialog box asks you to confirm that the user should be deleted.

  8. In the dialog box, click OK.
