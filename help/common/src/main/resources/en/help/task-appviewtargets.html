To View Application Targets

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. Select an application for which you want to view the targets.

    The application name can include an optional version identifier, which follows the name and is separated from the name by a colon (:). For more information about module and application versions, see the Module and Application Versions in Oracle GlassFish Server 3.1 Application Deployment Guide.

    The page for the application opens.

  3. Select the Targets tab.

    The Application Target page opens, and the targets currently specified for the application appear.

  4. To enable or disable an application on a specific target instance or cluster, click the checkbox next to the target and select Enable or Disable from the More Actions drop-down menu.
  5. To enable a target to be load-balanced on a specific target instance or cluster, click the checkbox next to the target and select Load Balancer Enable or Load Balancer Disable from the More Actions drop-down menu.
  6. To associate an application with specific virtual servers, select Manage Virtual Servers for the desired target instance or cluster.

    The Virtual Server Targets page opens.

    To add target virtual servers, select them from the Available Targets list and use the Add button to move them to the Selected Targets list.

    By default, the module or application is associated with the default virtual server, named server. Use the Add All button to add all targets. Use the Remove and Remove All buttons to remove targets from the Selected Targets list.

    Select Save to return to the Application Target page.

  7. To add or delete target instances or clusters for the application, select Manage Targets.

    The Manage Target page opens.

  8. To add or delete targets for the application, select the target from the Available Target list and click add, (or add all to add all targets), or remove (or remove all) to delete the targets.
  9. Click Save.

    The application is now available on the revised list of targets.
