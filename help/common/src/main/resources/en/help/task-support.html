To View the GlassFish News Page for GlassFish Server

The GlassFish News page provides a collection of links to additional information about the Oracle GlassFish Server, such as newsletters, articles, tech tips, discussion forums, and relevant blogs.

  1. In the navigation tree, select the Common Tasks node.

    The Common Tasks page opens.

  2. Under GlassFish News, click the GlassFish News link.

    The GlassFish News page opens in a separate browser tab or window.
