To View the GlassFish News Page for the Sun GlassFish Enterprise Server

The GlassFish News page provides a collection of links to additional information about the Sun GlassFish Enterprise Server, such as newsletters, articles, tech tips, discussion forums, and relevant blogs.

  1. In the navigation tree, select the GlassFish News node.

    The GlassFish News page opens.

  2. Click any link to open a page in a separate browser tab or window.
