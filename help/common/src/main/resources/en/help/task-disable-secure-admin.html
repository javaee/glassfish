To Disable Secure Administration

  1. In the navigation tree, select the server (Admin Server) node.

    The General Information page opens.

  2. On the General Information page, click the Secure Administration button.

    The Secure Administration page opens.

  3. Click Disable Secure Admin.

    A message appears, indicating that disabling secure administration will restart the DAS immediately.

  4. Click OK.

    Secure administration is disabled and the server restart page opens.
