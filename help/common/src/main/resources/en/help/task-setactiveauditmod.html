To Set the Active Audit Module

  1. In the navigation tree, expand the Configuration node.
  2. Under the Configuration node, select the Security node.

    The Security page opens.

  3. From the Audit Modules drop-down list, select the name of the audit module to be used by the server.

    By default, there is only one available audit module, named default.

  4. Click Save.
