Available Updates

The Available Updates page provides information about newer versions of components of the installation image that are installed.

The Available Updates page contains the following options.

Installation Path

The file system path of the root or top level directory of the installation image.

Install

This button updates the selected components.

Available Updates

This table provides the following information about each component for which a newer version is available:

Component

The name of the component.

Category

The category of software with which the component is associated, for example, Application Servers, System Tools, and Databases. The category is assigned to the component by the author of the component when the component is created.

Version

The version number of the package in which the component is supplied.

Installed Size

The amount of disk space that the installation of the component consumes.

Source

For an installed component, the name of the Network Repository from where the component was obtained.

For an add-on component or update, the name of the network repository from where the component would be obtained.
