Available Updates

The Available Updates page provides information about newer versions of components of the installation image that are installed.

To install available updates, first stop the GlassFish Server domain. Then use one of these commands:

  • glassfish3/bin/updatetool, which starts the Update Tool graphical utility

  • glassfish3/bin/pkg, a command-line version of the Update Tool

For information on using the pkg command, see Chapter 11, Extending and Updating GlassFish Server, in Oracle GlassFish Server Administration Guide. The Update Tool utility has its own online help.

Restart the server after you have finished installing the updates.

The Available Updates page contains the following options.

Installation Path

The file system path of the root or top level directory of the installation image.

Available Updates

For each component for which a newer version is available, this table provides the following information.

Component

The name of the component.

Category

The category of software with which the component is associated, for example, Application Servers, System Tools, and Databases. The category is assigned to the component by the author of the component when the component is created.

Installed Version

The version number of the package in which the current component is supplied.

New Version

The version number of the available update.

Change

Difference in size between the installed version and the available update.

Source

The name of the Network Repository from which the component would be obtained.
