Resources

Use the Resources page to configure the following resources for the selected GlassFish Server instance or cluster:

  • JDBC

  • Connectors

  • Resource Adapter Configs

  • JMS Resources

  • JavaMail Sessions

  • JNDI

The Resources page also contains the following option:

Add Resources

Button to create one or more resources from a glassfish-resources.xml file.
