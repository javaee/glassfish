Resource Targets

Use the Resource Targets page to view target clusters and standalone server instances on which the resource can be enabled.

The Resource Targets page contains the following information for each target.

Target Name

The name of the cluster or standalone server instance.

Enabled

Displays a check mark if the resource is enabled on the target, or an ex (X) if the resource is disabled.

The Targets table also contains the following options.

Manage Targets

Button to manage resource targets.

Enable

Button to enable the resource on the selected target.

Disable

Action to disable the resource on the selected target.
