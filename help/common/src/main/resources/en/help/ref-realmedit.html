Edit Realm

Use the Edit Realm page to modify a realm.

The Edit Realm page contains the following options.

Configuration Name

The name of the configuration to which the settings on this page apply. This field is read only.

Realm Name

The name of the realm. This is a read-only field. You can only specify the name when you create a new realm.

Class Name

The class name for the realm that you are editing.

Additional Properties

Additional properties for the realm. Valid properties depend on the type of realm.
