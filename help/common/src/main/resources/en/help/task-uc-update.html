To View Available Updates for Components

The Administration Console displays available updates for installed components. To update installed components, use one of these commands:

  • glassfishv3/bin/updatetool, which starts the Update Tool graphical utility

  • glassfishv3/bin/pkg, a command-line version of the Update Tool

For information on using the pkg command, see Chapter 10, Extending Enterprise Server, in Sun GlassFish Enterprise Server v3 Administration Guide. The Update Tool utility has its own online help.

  1. In the navigation tree, select the Update Tool node.

    The Available Add-Ons page opens.

  2. On the Available Add-Ons page, click the Available Updates tab.

    The Available Updates page opens. If no updates are available, the table is empty.
