To View the Subscriptions Page for GlassFish Server

  1. In the navigation tree, select the Common Tasks node.

    The Common Tasks page opens.

  2. Under GlassFish News, click the Subscriptions link.

    The GlassFish Subscriptions web page opens in a separate browser tab or window.
