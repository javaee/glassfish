Application Scoped Resources

Use the Application Scoped Resources page to view resources that are available only to a particular application.

The Application Scoped Resources page contains a table with the following information.

Application Name

The name of the application.

Module Name

The name of the module within the application.

Resource Name

The name of the application-scoped resource for the application or module. Click on the name to modify resource settings.

Resource Type

The resource type, for example JDBC Connection Pools or Connector Resources.
