Raw Log Viewer

Use the Raw Log Viewer page to view the raw log files for the Admin Server (named server).

The Raw Log Viewer page contains the following options.

Instance

The server instance or cluster for which log records will be displayed. The only choice is server.

For each message, the Raw Log Viewer displays the following information.

Timestamp

The date and time when the log message was issued.

Record Number

The number of the message in the server log.

Log Level

The log level of the message.

Version

The version of GlassFish Server.

Logger

The logger module that generated the log message.

Name/Value Pairs

The thread name and ID of the log message.

Message

The brief text of the log message. Click the (details) link to view a formatted version of a message.
