To View Descriptors of an Application

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. On the Applications page, click the name of the application whose descriptors you are viewing.

    The Edit Application page opens.

  3. On the Edit Application page, click the Descriptor tab.

    The Module Descriptors page opens.

  4. In the Deployment Descriptors table, click the name of the XML file you want to view.

    The XML file opens in a new window.
