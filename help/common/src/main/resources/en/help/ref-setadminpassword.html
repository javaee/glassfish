New Administrator Password

Use the New Administrator Password page to change the password for the current administrator account.

The New Administrator Password page contains the following options.

User ID

The user ID of the current administrator. This is a read-only field.

Group List

The group to which the administrator belongs. This a read-only field. The value is asadmin.

New Password

The new password for the administrator.

Confirm New Password

The new password for the administrator.
