To Add Resources Configured in an XML File

  1. In the navigation tree, select the Resources node.

    The Resources page opens.

  2. Click the Add Resources button.

    The Add Resources page opens.

  3. Select one of the following options to specify the location of the glassfish-resources.xml file:

    XML File to Be Uploaded to the Server

    Type the path to the file if it is located on a remote machine.

    Local XML File That Is Accessible from GlassFish Server

    Type the path to the file if it is located on the domain administration server (DAS) machine.

  4. From the Target drop-down list, select the cluster or stand-alone server instance on which to create the resources.

  5. Click OK.
