Update Tool Configuration

The Configuration page provides configuration information about Update Tool and enables you to change the configuration of Update Tool.

The Update Tool Configuration page contains the following options.

Installation Path

The file system path of the root or top level directory of the installation image.

Preferred Repository

The name of the preferred network repository that is associated with this installation image, for example, glassfish.org.

Internet Connection

How the GlassFish Server connects to the Internet:

Direct Connection

This option specifies that the GlassFish Server connects directly to the Internet.

Through a Proxy

This option specifies that the GlassFish Server connects to the Internet through a proxy server. When this option is selected, the Proxy Host field and the Proxy Port field are added to this page.

Proxy Host

The host name of the proxy server for the machine where the GlassFish Server is running.

This field is applicable only if the machine where the GlassFish Server is running connects to the Internet through a proxy server.

Proxy Port

The port number of the proxy server for the machine where the GlassFish Server is running.

This field is applicable only if the machine where the GlassFish Server is running connects to the Internet through a proxy server.

Repositories

This table provides the following information about each network repository that is associated with this installation image:

Repository

The name of the repository, for example, dev.glassfish.org.
