Manage Resource Targets

Use the Manage Resource Targets page to change the target clusters and standalone server instances on which a resource can be enabled. The resource can be enabled only on targets in the Selected Targets column.

The Manage Resource Targets page contains the following information.

Available Targets

The clusters and instances on which the resource cannot be enabled.

Selected Targets

The clusters and instances on which the resource can be enabled.

Add

Button to move the selected target from Available Targets to Selected Targets.

Add All

Button to move all Available Targets to Selected Targets.

Remove

Button to move the selected target from Selected Targets to Available Targets.

Remove All

Button to move all Selected Targets to Available Targets.
