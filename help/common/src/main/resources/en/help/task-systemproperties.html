To Manage System Properties

  1. In the navigation tree, select the Enterprise Server node.

    The General Information page opens.

  2. On the General Information page, click the System Properties tab.

    The System Properties page opens.

  3. In the Additional Properties section, specify system properties for the configuration.

    A number of system properties are predefined but are not shown on this page. For details, see Predefined System Properties.

    • To add a property, click the Add Property button. In the blank row that appears, type the property name in the Name field, and type the property value in the Value field.
    • To modify a property, edit that property's Value field.
    • To delete a property, select the checkbox to the left of the Name field of the property that you are deleting, then click the Delete Properties button.
  4. Click Save.
