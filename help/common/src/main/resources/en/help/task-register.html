To Register the Sun GlassFish Enterprise Server

As a registered user of the Sun GlassFishTM Enterprise Server, you receive benefits such as:

  • Patch information and bug updates

  • Screencasts and tutorials

  • News and events

  • Support and training offerings

To register your installation of Sun GlassFish Enterprise Server, you must have a Sun Online Account. If you do not have a Sun Online Account, the Enterprise Server enables you to create an account during registration.

Note - If you create a Sun Online Account while registering the Enterprise Server, you are prompted for only the required information. After registering the Enterprise Server, you can provide additional, optional information by updating your Sun Online Account.

Before You Begin

If the machine where the Enterprise Server is running connects to the Internet through a proxy, obtain the host name and port number of the proxy. If a web browser is running on the machine where the Enterprise Server is running, you can obtain this information from the web browser. Otherwise, ask your system administrator for this information.

  1. In the navigation tree, select the Registration node.

    The Product Registration page opens.

  2. In the Product Registration page, provide the required information.
    • If you already have a Sun Online Account, provide the required information as follows:
      1. Select the I Already Have a Sun Online Account or a Sun Developer Network (SDN) option.
      2. In the User Name field, type the user name that you specified when you registered for your Sun Online Account.
      3. In the Password field, type the password that you specified when you registered for your Sun Online Account.
    • If you do not have a Sun Online Account, provide the required information as follows:
      1. Select the I Don't Have a Sun Online Account option.
      2. In the Email Address field, type your email address.

        Your email address will be your Sun Online Account user name.

      3. In the Password field, type the password to use for your Sun Online Account.

        The password is case sensitive and must be 6–10 characters long.

      4. In the Retype Password field, retype the password that you typed in the preceding step.
      5. In the First Name field, type your first name.
      6. In the Last Name field, type your last name.
      7. In the Company field, type the name of your company, if applicable.
      8. From the drop down list, select the country or territory where you reside.
  3. If the machine where the Enterprise Server is running connects to the Internet through a proxy, provide the host name and port number of the proxy.

    If the machine where the Enterprise Server is running connects directly to the Internet, omit this step.

    A web browser might be running on the machine where the Enterprise Server is running. In this situation, you can obtain the host name and port number of the proxy from the web browser. Otherwise, ask your system administrator for this information.

  4. Click Register.

    To confirm that the registration is complete, the Support page opens.
