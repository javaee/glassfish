Domain Logs

Use the Domain Logs page to collect and download log files for the GlassFish Server.

The Domain Logs page contains the following options. You can select one cluster or one server instance for which to collect logs.

Cluster

A drop-down list of clusters if any clusters exist.

Instance

A drop-down list of server instances.

Collect Logs

Button to collect the logs into a log-files-name.zip file, where name is the name of the cluster or instance.

The log-files-name.zip file is downloaded using your browser's download mechanism.
