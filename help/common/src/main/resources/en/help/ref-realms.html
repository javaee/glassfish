Realms

The Enterprise Server comes preconfigured with three realms: file, certificate, and admin-realm. It is also possible to create ldap, JDBC, solaris, and custom realms. Generally, you will have one realm of each type on a server, but on the Enterprise Server there are two file realms: file and admin-realm. These are two realms of the same type used for two different purposes. It is also possible to have a different certificate database for each virtual server on your system.

The Realms page displays a list of available realms. For each realm, the following information is provided:

  • Name

  • Class Name
