To Restart a Deployed Application

Restarting an application simply restarts the application without redeploying it, preserving the settings and other options with which the application was deployed.

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. In the Deployed Applications table, click Restart in the row for the application that you are redeploying.

    The Applications page appears again, with a message stating whether the restart succeeded.
