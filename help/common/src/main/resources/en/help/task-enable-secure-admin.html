To Enable Secure Administration

  1. In the navigation tree, select the server (Admin Server) node.

    The General Information page opens.

  2. On the General Information page, click the Secure Administration button.

    The Secure Administration page opens.

  3. If the installation of GlassFish Server uses customized administration or instance alias values, update the default values shown in the Administration Alias and Instance Alias fields as needed.

  4. Click Enable Secure Admin.

    A message appears, indicating that enabling secure administration will restart the DAS immediately.

  5. Click OK.

    Secure administration is enabled and the server restart page opens.
