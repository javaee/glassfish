Properties Specific to JACC Providers

The following property is available for a JACC provider.

repository

The directory containing the JACC policy file. For the defaultGlassFish Server JACC provider, the default directory is ${com.sun.aas.instanceRoot}/generated/policy. This property is not defined by default for the simpleGlassFish Server JACC provider.
