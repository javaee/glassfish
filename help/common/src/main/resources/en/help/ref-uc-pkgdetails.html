Component Details

The Component Details page provides the detailed information about a component.

The Component Details page contains the following options.

Back

This button returns you to the page from which you selected the component whose details you are viewing.

Package Name

The name of the package in which the component is supplied.

Unique Identifier

The fully qualified identifier of this component as it exists in the repository.

Category

The category of software with which the component is associated, for example, Application Servers, System Tools, and Databases. The category is assigned to the component by the author of the component when the component is created.

Version

The version number of the package in which the component is supplied.

Date Published

The date and time when this version of the component was published to the repository. This date is not the date and time when the component was installed within the installation image.

Installed Size

The amount of disk space that the installation of the component consumes.

Source

For an installed component, the name of the Network Repository from where the component was obtained.

For an add-on component or update, the name of the network repository from where the component would be obtained.

Description

An overview of the component.
