Edit File Realm User

Use the Edit File Realm User page to change a user's password for the current realm.


Note:

If secure administration is enabled, you cannot change the password of an admin-realm user to an empty value.


The Edit File Realm User page contains the following options.

Configuration Name

The name of the configuration to which the settings on this page apply. This field is read only.

Realm Name

The name of the currently selected realm. This is a read-only field.

User ID

The name of the account that you are editing. This is a read-only field.

Group List

Groups to which this user belongs. Separate group names with commas.

For admin-realm users, this is a read-only field, and the group is asadmin.

New Password

New password for this user.

Confirm New Password

New password for this user.
