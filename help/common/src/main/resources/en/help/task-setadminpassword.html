To Change the Administrator Password

  1. In the navigation tree, select the Enterprise Server node.

    The General Information page opens.

  2. Select the Administrator Password tab.

    The New Administrator Password page opens.

    The page displays the user ID and group name of the current administrator account. By default, the user ID is admin, and the group name is asadmin.

  3. In the New Password field, type the new password for the current administrator account.
  4. In the Confirm New Password field, type the new password again.
  5. Click Save.
