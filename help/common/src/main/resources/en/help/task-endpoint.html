To View Endpoint Information for a Web Service

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. On the Applications page, click the name of the web service whose endpoint information you are viewing.

    The name can include an optional version identifier, which follows the name and is separated from the name by a colon (:). For more information about module and application versions, see the Oracle GlassFish Server 3.1 Application Deployment Guide.

    The Edit Application page opens.

  3. In the Modules and Components table, click the View Endpoint link for the subcomponent.

    The Web Service Endpoint Information page opens.

  4. If the Tester link is visible, click this link to view a page that allows you to supply parameters to test the methods of the web service.

    The Tester link is visible only if the application's Deployment Type is other than RI (for example, if it is 109). The Tester page opens in a new browser tab or window.

  5. Click the WSDL link to view the Web Services Definition Language (WSDL) file for the endpoint.

    The WSDL file opens in a new browser tab or window.

  6. Click the Application Name link or the Back button to return to the Edit Application page.
