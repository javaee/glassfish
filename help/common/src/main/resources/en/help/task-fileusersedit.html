To Edit File Realm User Information

  1. In the navigation tree, expand the Configuration node.
  2. Expand the Security node.
  3. Expand the Realms node.
  4. Select the File node.

    The Edit Realm page opens.

  5. On the Edit Realm page, click the Manage Users button.

    The File Users page opens.

  6. On the File Users page, in the User ID column of the table, click the name of the user to be modified.

    The Edit File Realm User page opens.

  7. Edit the Group List field to modify the groups to which the user belongs.

    Use commas to separate the group names. These groups do not need to be defined elsewhere.

  8. Type a new password in the New Password and Confirm New Password fields to change the user's password.
  9. Click Save.
