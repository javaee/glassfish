To Launch an Application

Before You Begin

Ensure that the application you are launching is deployed.

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. In the Deployed Applications table, click Launch in the row for the application that you are launching.

    The application opens in a new browser window.
