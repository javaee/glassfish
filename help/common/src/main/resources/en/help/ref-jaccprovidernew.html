New JACC Provider

Use the New JACC Provider page to define a new JACC provider.

The New JACC Provider page contains the following options.

Name

The name that will identify this provider.

Policy Configuration

The name of the class that implements the policy configuration factory.

Policy Provider

The name of the class that implements the policy factory.

Additional Properties

Additional properties for the JACC provider. For a description of the additional properties that are available for JACC providers, see Properties Specific to JACC Providers.
