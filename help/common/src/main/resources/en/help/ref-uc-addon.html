Available Add-Ons

The Available Add-Ons page provides information about components of the installation image that are not yet installed.

The Available Add-Ons page contains the following options.

Installation Path

The file system path of the root or top level directory of the installation image.

Install

Button to install one or more selected add-on components.

Available Add-Ons

For each component of the installation image that is not yet installed, this table provides the following information.

Component

The name of the component.

Category

The category of software with which the component is associated, for example, Application Servers, System Tools, and Databases. The category is assigned to the component by the author of the component when the component is created.

Version

The version number of the package in which the component is supplied.

Install Size

The amount of disk space that the installation of the component consumes.

Source

The name of the Network Repository from which the component would be obtained.
