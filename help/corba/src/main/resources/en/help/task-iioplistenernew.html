To Create an IIOP Listener

  1. In the tree component, expand the Configuration node.
  2. Under the Configuration node, expand the ORB node.
  3. Select IIOP Listeners.

    The IIOP Listeners page opens.

  4. Click New.
  5. In the Name field, type a name to identify the listener.
  6. In the Network Address field, type the network address of the listener.

    You can type an IP address or a DNS resolvable host name.

  7. In the Listener Port field, type the port number upon which the IIOP listener is to listen.
  8. Select the Listener Enabled checkbox to enable ORB IIOP inbound connections to the Enterprise Server.

    This option is enabled by default.

  9. Select the Security Enabled checkbox to enable SSL for the IIOP listener.

    This option is disabled by default.

  10. (Optional) In the Additional Properties section, specify additional properties.

    The Enterprise Serverdoes not define any additional properties for IIOP listeners.

    To add a property, click the Add Property button. In the blank row that appears, type the property name in the Name field, and type the property value in the Value field.

  11. Click OK.
  12. If you selected the Security Enabled checkbox, follow the steps in To Edit SSL Settings for a Secure IIOP Listener.
