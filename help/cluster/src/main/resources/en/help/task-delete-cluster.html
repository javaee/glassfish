To Delete a Cluster

Deleting a cluster deletes the cluster and all GlassFish Server instances in the cluster. You are not required to delete the instances in the cluster before deleting the cluster.

If the configuration that the cluster uses is not shared, the configuration is also deleted.

Before You Begin

Ensure that the cluster that you are deleting is stopped.

  1. In the navigation tree, select the Clusters node.

    The Clusters page opens.

  2. On the Clusters page, select the checkbox adjacent to the name of the cluster.
  3. Click Delete.

    You are asked to confirm that you want to delete the cluster and all instances in the cluster.

  4. Confirm that you want to delete the cluster and all instances in the cluster.

    The cluster is deleted and is removed from the table of clusters on the Clusters page.
