To View Log Files for a Standalone GlassFish Server Instance

Before You Begin

The instance for which you are viewing log files must be running.

  1. In the navigation tree, select the Standalone Instances node.

    The Standalone Server Instances page opens.

  2. Select the instance for which to view log files.

    The General Information page for the instance opens.

  3. On the General Information page, click View Log Files.

    A Log Viewer window opens.


    Note:

    The instance must already be running. If the instance is not running, the View Log Files button is deactivated.
