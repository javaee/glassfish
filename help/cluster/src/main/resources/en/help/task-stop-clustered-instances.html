To Stop Clustered GlassFish Server Instances

  1. In the navigation tree, expand the Clusters node.
  2. Under the expanded Clusters node, select the node for the cluster.

    The General Information page for the cluster opens.

  3. On the General Information page, click the Instances tab.

    The Clustered Server Instances page for the cluster opens.

  4. For each instance that you are stopping, select the check box adjacent to the instance in the Server Instances table.
  5. Click Stop.

    You are asked to confirm that you want to stop the instances.

  6. Confirm that you want to stop the instances.

    The status of each instance is updated in the Server Instances table when the instance is stopped.
