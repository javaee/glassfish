To Create a Resource for a Standalone GlassFish Server Instance

  1. In the navigation tree, select the Standalone Instances node.

    The Standalone Server Instances page opens.

  2. On the Standalone Server Instances page, select the instance.

    The General Information page for the instance opens.

  3. On the General Information page, click the Resources tab.

    The Resources page for the instance opens.

  4. From the New pull-down menu, select the type of the resource to create.

    The page for creating a resource of the selected type opens.

  5. On the page for creating a resource of the selected type, make the required changes and save your changes.

    For details, see the online help for the page for creating a resource of the selected type.

    Note - When you create the resource, ensure that you specify the instance as a target.
