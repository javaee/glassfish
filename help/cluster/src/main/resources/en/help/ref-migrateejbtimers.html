Migrate EJB Timers

The Migrate EJB Timers page contains the following options.

Source

Drop-down list of stopped GlassFish Server instances in the cluster. EJB timers can be migrated only from instances that are stopped.

Destination

Drop-down list of running GlassFish Server instances in the cluster. EJB timers can be migrated only to instances that are running.

Note - If all instances in the cluster are stopped, this page displays only a warning that no running instances to which EJB timers can be migrated are available.
