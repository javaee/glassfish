To Deploy an Application to a Cluster

  1. In the navigation tree, expand the Clusters node.

  2. Under the expanded Clusters node, select the node for the cluster.

    The General Information page for the cluster opens.

  3. On the General Information page, click the Applications tab.

    The Applications page for the cluster opens.

  4. On the Applications page, click Deploy.

    The Deploy Applications or Modules page opens.

  5. On the Deploy Applications or Modules page, provide the required information about the application and click OK.

    For details, see the Deploy Applications or Modules online help.
