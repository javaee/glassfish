Availability Service

Use the Availability Service page to change availability settings for a cluster.

The Availability Service page contains the following options.

Load Defaults

Button to restore the default values of all the settings.

Availability Service

If the Enabled checkbox is selected, availability is enabled for the cluster. This option is selected by default.

Additional Properties

Additional properties for the Availability Service.

The GlassFish Server does not define any additional Availability Service properties.
