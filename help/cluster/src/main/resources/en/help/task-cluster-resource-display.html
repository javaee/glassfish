To List Resources for a Cluster

  1. In the navigation tree, expand the Clusters node.
  2. Under the expanded Clusters node, select the node for the cluster.

    The General Information page for the cluster opens.

  3. On the General Information page, click the Resources tab.

    The Resources page for the cluster opens.

  4. (Optional) To display only resources of a particular type, from the Filter pull-down list, select the type of resource to display.
