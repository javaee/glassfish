To Delete a JavaMail Session

  1. In the navigation tree, expand the Resources node .
  2. Under the Resources node, select JavaMail Sessions.

    The JavaMail Sessions page opens.

  3. On the JavaMail Sessions page, select the checkbox next to the name of the session that you are deleting.
  4. Click Delete.
