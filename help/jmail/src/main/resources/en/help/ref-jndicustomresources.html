Custom Resources

A custom resource specifies a custom server-wide resource object factory.

Use the Custom Resources page to configure custom JNDI resources.

For each custom resource, the Custom Resources page contains the following information.

JNDI Name

A unique name that identifies the custom resource.

Enabled

True if the resource is enabled, or false if the resource is disabled.

Resource Type

The type of the resource.

Description

A description of the resource.
