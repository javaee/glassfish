To Manage JavaMail Session Targets

  1. In the navigation tree, expand the Resources node .

  2. Under the Resources node, select JavaMail Sessions.

    The JavaMail Sessions page opens.

  3. On the Sessions table, click the session name whose targets you want to manage.

    The Edit JavaMail Session page opens.

  4. Click the Target tab.

    The JavaMail Session Targets page opens.

  5. In the Targets table, choose the action you want to take:

    • To add or remove targets, click Manage Targets.

    • To enable a target, click the checkbox next to the desired server instance or cluster and click Enable.

    • To disable a target, click the checkbox next to the desired server instance or cluster and click Disable.
