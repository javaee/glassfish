External Resources

An external JNDI resource accesses an external JNDI repository, such as an LDAP server.

Use the External Resources page to configure external JNDI resources.

For each external resource, the External Resources page contains the following information.

JNDI Name

A unique name that identifies the external resource.

Enabled

True if the resource is enabled, or false if the resource is disabled.

Resource Type

The type of the resource.

Description

A description of the resource.
