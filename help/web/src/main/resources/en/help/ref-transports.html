Transports

Use the Transports page to configure transports. Each network listener is associated with a transport.

When you first install the Enterprise Server, one TCP transport exists by default, with the name tcp. You can modify or create a TCP or UDP transport or provide a custom transport implementation.

For each transport, the Transports page contains the following information.

Transport Name

The name of the transport.

Classname

The class name of the transport implementation.

Byte Buffer Type

The type of ByteBuffer to be used with the transport.
