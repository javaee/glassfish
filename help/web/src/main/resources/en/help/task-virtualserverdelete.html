To Delete a Virtual Server

  1. In the navigation tree, expand the Configuration node.
  2. Under the Configuration node, select the Virtual Servers node.

    The Virtual Servers page opens.

  3. On the Virtual Servers page, select the checkbox to the left of the name of the virtual server that you are deleting.
  4. Click Delete.

    Deleting either of the default virtual servers, __asadmin and server, is not recommended.
