To Edit a Protocol

  1. In the navigation tree, expand the Configuration node.
  2. Under the Configuration node, expand the Network Config node.
  3. Under the Network Config node, select the Protocols node.

    The Protocols page opens.

  4. On the Protocols page, click the name of the protocol that you are editing.

    The Edit Protocol page opens.

  5. Select the Security Enabled checkbox to enable security for the protocol.

    If security is enabled, you can use the SSL tab to modify the security settings.

  6. Click Save.

    Tip - To return to the default settings, click the Load Defaults button, then click Save.

  7. If the Security Enabled checkbox is selected, follow the steps in To Edit SSL Settings for a Protocol.
