Network Listeners

Use the Network Listeners page to configure network listeners.

When you first install the Enterprise Server, three network listeners exist by default, with the names admin-listener, http-listener1, and http-listener2. The http-listener2, which is configured with SSL, is disabled by default.

For each network listener, the Network Listeners page contains the following information.

Name

The name of the network listener.

Port #

The port number on which the network listener is listening.

Protocol

The protocol used by the network listener.

Thread Pool

The thread pool used by the network listener.

Enabled

True if the network listener is enabled, or false if the network listener is not enabled.
