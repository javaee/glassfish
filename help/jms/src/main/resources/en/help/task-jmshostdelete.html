To Delete a JMS Host

Deleting the default JMS host is not recommended.

  1. In the navigation tree, expand the Configuration node.
  2. Under the Configuration node, expand the Java Message Service node.
  3. Under the Java Message Service node, select the JMS Hosts node.

    The JMS Hosts page opens.

  4. On the JMS Hosts page, select the checkbox next to the name of the host that you are deleting.

    Caution - Do not delete the default JMS host. Doing so can lead to unexpected errors.

  5. Click Delete.
