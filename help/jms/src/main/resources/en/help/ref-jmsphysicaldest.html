JMS Physical Destinations

Java Message Service (JMS) physical destination objects are maintained by Message Queue brokers. The queue named mq.sys.dmq is the system destination, to which expired and undeliverable messages are redirected.

Use the JMS Physical Destinations page to configure Message Queue physical destinations.

For each physical destination, the JMS Physical Destinations page contains the following information.

Name

The name of the physical destination.

Type

The type of the physical destination.

Statistics

Statistics for the physical destination.

Flush

Button to remove messages from a physical destination.
