JMS Availability

Use the JMS Availability page to configure availability settings for the Java Message Service (JMS).

The JMS Availability page contains the following options.

Load Defaults

Button to restore the default values of all the settings.

Configuration Name

The name of the configuration to which the settings on this page apply. This field is read only.

Availability Service

If the Enabled checkbox is selected, availability is enabled for the JMS.

MQ Store Pool Name

The JNDI name of the JDBC resource that the GlassFish Message Queue broker cluster is to use when saving persistent JMS messages and other broker configuration information.

The pool name can contain only letters, digits or underscore (_), dash (-) or period (.) characters.

Additional Properties

Additional properties for JMS availability.

GlassFish Server does not predefine any additional JMS availability properties.
