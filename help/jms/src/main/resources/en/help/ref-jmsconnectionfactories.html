JMS Connection Factories

Use the JMS Connection Factories page to configure JMS connection factory resources.

For each resource, the JMS Connection Factories page contains the following information.

JNDI Name

A unique name that identifies the connection factory.

Enabled

True if the connection factory is enabled, or false if the connection factory is disabled.

Resource Type

The resource type of the connection factory.

Description

A description of the connection factory.
