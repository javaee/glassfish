To Delete a JDBC Connection Pool

When you delete a JDBC connection pool, any JDBC resources that depend on that connection pool are also deleted.

  1. In the navigation tree, expand the Resources node.

  2. Under the Resources node, expand the JDBC node.

  3. Under the JDBC node, select the Connection Pools node.

    The Connection Pools page opens.

  4. On the Connection Pools page, select the checkbox to the left of the connection pool that you are deleting.

  5. Click Delete.
