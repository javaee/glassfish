To Edit a JDBC Resource

  1. In the navigation tree, expand the Resources node.

  2. Under the Resources node, expand the JDBC node.

  3. Under the JDBC node, select the JDBC Resources node.

    The JDBC Resources page opens.

  4. On the JDBC Resources page, click the name of the resource to be edited.

    The Edit JDBC Resource page opens.

  5. From the Pool Name drop-down list, select a connection pool to be associated with this JDBC resource.

  6. In the Description field, type a short description for this JDBC resource.

  7. In the Additional Properties section, specify additional properties.

    The GlassFish Serverdoes not define any additional properties for JDBC resources.

    • To add a property, click the Add Property button. In the blank row that appears, type the property name in the Name field, and type the property value in the Value field.

    • To modify a property, edit that property's Value field.

    • To delete a property, select the checkbox to the left of the Name field of the property that you are deleting, then click the Delete Properties button.

  8. Click the Target tab to manage Resource Targets.

    1. Enable or disable defined targets for the resource, as desired.

    2. Click the Manage Targets button to add or remove targets for the resource.

    See To Edit JDBC Resource Targets for more information.

  9. Click the General tab to return to the Edit JDBC Resource page.

  10. Click Save.
