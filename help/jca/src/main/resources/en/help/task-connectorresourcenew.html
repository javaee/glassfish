To Create a Connector Resource

Before You Begin

Before you create a connector resource, first create a connector connection pool.

  1. In the navigation tree, expand the Resources node.
  2. Under the Resources node, expand the Connectors node.
  3. Under the Connectors node, select the Connector Resources node.

    The Connector Resources page opens.

  4. On the Connector Resources page, click the New button.

    The New Connector Resource page opens.

  5. In the JNDI Name field, type a unique JNDI name for the connector resource.

    For example: eis/myERP.

  6. From the Pool Name drop-down list, select the connection pool to which the new connector resource belongs.
  7. In the Description field, type a description of the connector resource.
  8. Select the Status Enabled checkbox to enable the connector resource at run time.

    This option is enabled by default.

  9. Under Available Targets, select the domain, cluster, or server instances where the connector resource will reside and click Add.

    If you do not want to deploy the connector resource to one of the domains, clusters, or server instances listed under Selected Targets, select the target from the field and click Remove.

  10. (Optional) In the Additional Properties section, specify additional properties for the connector resource.

    Available properties depend upon the resource adapter for the connector connection pool.

    To add a property, click the Add Property button. In the blank row that appears, type the property name in the Name field, and type the property value in the Value field.

  11. Click OK.
