To Delete a Connector Resource

A connector resource is deleted automatically if you delete the connector connection pool associated with the resource.

  1. In the navigation tree, expand the Resources node.

  2. Expand the Connectors node.

  3. Under the Connectors node, select the Connector Resources node.

    The Connector Resources page opens.

  4. On the Connector Resources page, select the checkbox next to the name of the resource that you are deleting.

  5. Click Delete.
