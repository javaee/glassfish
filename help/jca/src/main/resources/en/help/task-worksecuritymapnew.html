To Create a Work Security Map

  1. In the navigation tree, expand the Resources node.
  2. Expand the Connectors node.
  3. Under the Connectors node, select the Work Security Maps node.

    The Work Security Maps page opens.

  4. On the Work Security Maps page, click New.

    The New Work Security Map page opens.

  5. In the Security Map Name field, type a name for the work security map.
  6. Use either the Group fields or the Principal fields as follows.
    • If you select the Group Mapping radio button:
      1. In the field labeled EIS Group, type the name of the back-end EIS user group.
      2. In the field labeled Mapped Group, type the name of the corresponding user group defined in the Enterprise Server security domain.
    • If you select the Principal Mapping radio button:
      1. In the field labeled EIS Principal, type the name of the back-end EIS principal.
      2. In the field labeled Mapped Principal, type the name of the corresponding principal defined in the Enterprise Server security domain.
  7. From the Resource Adapter drop-down list, select the resource adapter for the EIS.
  8. In the Description field, type a description of the work security map.
  9. Select the Status Enabled checkbox to enable the work security map.

    This option is enabled by default.

  10. Click OK.
