New Work Security Map

Use the New Work Security Map page to create a new work security map.

The New Work Security Map contains the following options.

Security Map Name

The name of the work security map.

Group

If the Group Mapping radio button is selected, the EIS Group text field must contain the name of the back-end EIS user group, and the Mapped Group text field must contain the name of the corresponding user group defined in the Enterprise Server security domain.

Principal

If the Principal Mapping radio button is selected, the EIS Principal text field must contain the name of the back-end EIS principal, and the Mapped Principal text field must contain the name of the corresponding principal defined in the Enterprise Server security domain.

Resource Adapter

The resource adapter for the EIS.

Description

A description of the work security map.

Status

If this option is selected, the work security map is enabled. This option is enabled by default.
