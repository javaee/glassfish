To Delete a Resource Adapter Config

  1. In the navigation tree, expand the Resources node.
  2. Under the Resources node, select the Resource Adapter Configs node.

    The Resource Adapter Configs page opens.

  3. On the Resource Adapter Configs page, select the checkbox next to the name of the resource adapter config that you are deleting.
  4. Click Delete.
