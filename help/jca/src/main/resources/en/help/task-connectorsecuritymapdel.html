To Delete a Security Map for a Connector Connection Pool

  1. In the navigation tree, expand the Resources node.
  2. Expand the Connectors node.
  3. Under the Connectors node, select the Connector Connection Pools node.

    The Connector Connection Pools page opens.

  4. On the Connector Connection Pools page, click the name of the connection pool that has the security map that you are deleting.

    The Edit Connector Connection Pool page opens.

  5. Click the Security Maps tab.

    The Security Maps page opens.

  6. On the Security Maps page, select the checkbox next to the name of the security map that you are deleting.
  7. Click Delete.
