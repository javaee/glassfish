To Delete a Connector Connection Pool

When you delete a connector connection pool, any connector resources that depend on that connector connection pool are also deleted.

  1. In the navigation tree, expand the Resources node.
  2. Expand the Connectors node.
  3. Under the Connectors node, select the Connector Connection Pools node.

    The Connector Connection Pools page opens.

  4. On the Connector Connection Pools page, select the checkbox next to the name of the connection pool that you are deleting.
  5. Click Delete.
