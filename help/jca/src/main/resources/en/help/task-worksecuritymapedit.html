To Edit a Work Security Map

  1. In the navigation tree, expand the Resources node.
  2. Expand the Connectors node.
  3. Under the Connectors node, select the Work Security Maps node.

    The Work Security Maps page opens.

  4. On the Work Security Maps page, click the name of the work security map that you are editing.

    The Edit Work Security Map page opens.

  5. Use either the Group fields or the Principal fields as follows.
    • If you select the Group Mapping radio button:
      1. In the field labeled EIS Group, type the name of the EIS group.
      2. In the field labeled Mapped Group, type the name of the corresponding group defined in the Enterprise Server domain.
    • If you select the Principal Mapping radio button:
      1. In the field labeled EIS Principal, type the name of the EIS principal.
      2. In the field labeled Mapped Principal, type the name of the corresponding principal defined in the Enterprise Server domain.
  6. From the Resource Adapter drop-down list, select the resource adapter for the EIS.
  7. In the Description field, type a description of the work security map.
  8. Select the Status Enabled checkbox to enable the work security map.

    This option is enabled by default.

  9. Click OK.
