To Install the JRuby Add-On Component

  1. In the navigation tree, select the Update Tool node.

    The Available Add-Ons page opens.

  2. On the Available Add-Ons page, select the checkbox to the left of the jruby add-on component and click the Install button.

    A confirmation dialog box opens.

  3. In the confirmation dialog box, click OK.

    The License Agreement page opens.

  4. On the License Agreement page, read the license agreement, then click the Accept button.

    The Accept button text changes to “Processing...”. If the installation succeeds, a success message appears on the Available Add-Ons page.

  5. To verify that installation succeeded, click the Installed Components tab.

    The jruby and jruby-gems components both appear in the table.
