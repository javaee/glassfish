To Edit a Context Service

  1. In the navigation tree, expand the Resources node.

  2. Under the Resources node, expand the Concurrent Resources node.

  3. Under the Concurrent Resources node, select Context Services.

    The Context Services page opens.

  4. On the Context Services page, click the name of the resource that you are editing.

    The Edit Context Service page opens.

    Note:

    The JNDI name is a read-only field. You can only specify the JNDI name when you create a new resource.

  5. XXX

  6. In the Description field, Type a short description of the resource.

    This description is a string value and can include a maximum of 250 characters.

  7. Select the Status Enabled checkbox to enable the resource.

  8. In the Additional Properties section, specify additional properties for the resource.

    GlassFish Server does not define any additional properties for context service resources.

    • To add a property, click the Add Property button. In the blank row that appears, type the property name in the Name field, and type the property value in the Value field.

    • To modify a property, edit that property's Value field.

    • To delete a property, select the checkbox to the left of the Name field of the property that you are deleting, then click Delete Properties.

  9. Click Save.
