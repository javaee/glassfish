Concurrent Resources

Use the Concurrent Resources pages to configure concurrent resources of the following types:

  • Context Services

  • Managed Executor Services

  • Managed Scheduled Executor Services

  • Managed Thread Factories

Concurrent resources are JSR 236 managed objects that provide concurrency capabilities to Java EE applications.

In the GlassFish Server, you configure an instance of a managed object and make it available for use by application components such as servlets and EJBs. Managed objects are accessed through Java Naming and Directory Interface (JNDI) lookup or resource injection.
