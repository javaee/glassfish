License Agreement

The License Agreement page displays the license agreements of all components that are selected for installation or update. The license agreement for a component is provided by the component author with the component. Different components in the Network Repository might be published under different licenses, for example:

  • Common Development and Distribution License (CDDL) (http://www.sun.com/cddl/)

  • GNU General Public License (GPL) (http://www.gnu.org/copyleft/gpl.html)

  • Apache License (http://www.apache.org/licenses/)

  • wxWindows Library Licence (http://www.opensource.org/licenses/wxwindows.php)

The License Agreement page contains the following options.

Accept

This button accepts the terms of the licenses of all selected components and proceeds with the installation or upgrade.

Decline

This button declines the terms of the licenses of all selected components and returns you to the page from which you selected the components.
