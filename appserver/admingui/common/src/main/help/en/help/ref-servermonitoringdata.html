Server Monitoring

Use the Server Monitoring page to view monitoring data for the server instance.

The Server Monitoring page contains the following information.

Instance Name

The name of the server instance.

View

The virtual server, HTTP listener, garbage collector, or thread system for which monitoring statistics are displayed.

For a description of the monitored properties, see Chapter 8, Administering the Monitoring Service, in Oracle GlassFish Server Administration Guide.
