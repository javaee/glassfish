Product Registration

To register your installation of Oracle GlassFish Server, you must have a Sun Online Account. If you do not have a Sun Online Account, the GlassFish Server enables you to create an account during registration.

The Product Registration page contains the following options.

I already have a Sun Online Account or a Sun Developer Network (SDN) Account

Select if you already have an account. If you select this option, the following fields are visible.

User Name

The user name that you specified when you registered for your Sun Online Account.

Password

The password that you specified when you registered for your Sun Online Account.

Proxy Host

The host name of the proxy server for the machine where the GlassFish Server is running.

This field is applicable only if the machine where the GlassFish Server is running connects to the Internet through a proxy server.

Proxy Port

The port number of the proxy server for the machine where the GlassFish Server is running.

This field is applicable only if the machine where the GlassFish Server is running connects to the Internet through a proxy server.

I don't have a Sun Online Account. Sign me up.

Select if you do not have an account and you want to register your installation of GlassFish Server. If you select this option, the following fields are visible.

Email Address

Your email address, which will be your Sun Online Account user name.

Password

The password to use for your Sun Online Account. The password is case sensitive and must be 6–10 characters long.

Retype Password

You must retype the password you specified in the Password field.

First Name

Your first name.

Last Name

Your last name.

Company

The company you work for, if applicable.

Country/Territory

The country or territory where you reside.

Proxy Host

The host name of the proxy server for the machine where the GlassFish Server is running.

This field is applicable only if the machine where the GlassFish Server is running connects to the Internet through a proxy server.

Proxy Port

The port number of the proxy server for the machine where the GlassFish Server is running.

This field is applicable only if the machine where the GlassFish Server is running connects to the Internet through a proxy server.
