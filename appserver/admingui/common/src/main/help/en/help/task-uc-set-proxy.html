To Configure Network Proxy Settings for Accessing Network Repositories

Network proxy settings determine how GlassFish Server connects to the Internet. To enable Update Tool to access network repositories through the Internet, configure your network proxy settings.

  1. In the navigation tree, select the Update Tool node.

    The Available Add-Ons page opens.

  2. On the Available Add-Ons page, click the Configuration tab.

    The Update Tool Configuration page is displayed.

  3. On the Update Tool Configuration page, specify how the machine where the GlassFish Server is running connects to the Internet.

    • If the machine connects to the Internet directly, select the Direct Connection option.

    • If the machine connects to the Internet through a proxy, select the Through a Proxy option.

  4. If the machine where the GlassFish Server is running connects to the Internet through a proxy, provide the Proxy Host name and Proxy Port number for the proxy.

    If the machine where the GlassFish Server is running connects directly to the Internet, omit this step.

    A web browser might be running on the machine where the GlassFish Server is running. In this situation, you can obtain the host name and port number of the proxy from the web browser. Otherwise, ask your system administrator for this information.

  5. Click Save.
