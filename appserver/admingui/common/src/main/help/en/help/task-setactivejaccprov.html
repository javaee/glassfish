To Set the Active JACC Provider

  1. In the navigation tree, expand the Configuration node.
  2. Under the Configuration node, select the Security node.

    The Security page opens.

  3. From the JACC drop-down list, select the JACC provider to be used by the server.

    By default, two JACC providers are available, with the names default and simple. The default provider is default.

  4. Click Save.
