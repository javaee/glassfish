To View General Information for the DAS

  1. In the navigation tree, select the server (Admin Server) node.

    The General Information page opens.

  2. On the General Information page, perform any of the following steps.

    • Look at the read-only Status display to see if the server is running.

    • Look at the read-only Uptime display to see the number of hours and minutes that the server has been continuously running.

    • Click the Stop button to stop the GlassFish Server.

    • Click the Restart button to stop and restart the GlassFish Server.

    • Click the View Log Files button to view log files for GlassFish Server instances or clusters.

      See To View Log Files for the DAS for details.

    • Click JVM Report to open a separate window that allows you to view reports on the Java Virtual Machine, including a summary report, memory management report, class loading report, and a current thread dump.
