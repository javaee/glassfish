To Stop a Cluster

Stopping a cluster stops all running GlassFish Server instances in the cluster.

  1. In the navigation tree, select the Clusters node.

    The Clusters page opens.

  2. On the Clusters page, select the checkbox adjacent to the name of the cluster.

  3. Click Stop Cluster.

    You are asked to confirm that you want to stop the cluster.

  4. Confirm that you want to stop the cluster.

    The cluster is stopped. The status of each instance in the cluster is updated on the Clusters page when the instance is stopped.
