To Set Instance-Only Properties for a Clustered GlassFish Server Instance

Properties of a GlassFish Server instance add optional configuration information about the instance.

  1. In the navigation tree, expand the Clusters node.

  2. Under the expanded Clusters node, select the node for the cluster.

    The General Information page for the cluster opens.

  3. On the General Information page, click the Instances tab.

    The Clustered Server Instances page for the cluster opens.

  4. Select the instance for which to modify instance-only properties.

    The General Information page for the instance opens.

  5. On the General Information page, click the Properties tab.

  6. If necessary, click the Instance Properties tab.

    The Instance Properties page opens.

    This page displays a list of the properties that are set for the selected GlassFish Server instance.

  7. For each existing property that you are modifying, type the new value in the Value field for the property.

  8. Set values for any additional properties that you require.

    1. In the Additional Properties table, click Add Property.

      A row is added to the Additional Properties table.

    2. In the Name field, type the property name.

    3. In the Value field, type the value.

    4. In the Description field, type a brief description of the property.

  9. Click Save.
