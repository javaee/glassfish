EJB Container Availability

Use the EJB Container Availability page to configure availability settings for the EJB container.

The EJB Container Availability page contains the following options.

Load Defaults

Button to restore the default values of all the settings.

Availability Service

If the Enabled checkbox is selected, availability is enabled for the EJB container.

HA Persistence Type

The persistence mechanism used when Availability Service Enabled is checked. Persistence is based on failover in a cluster. Do not change this value.

SFSB Persistence Type

The persistence mechanism used when Availability Service Enabled is not checked. Persistence is based on the file system. Do not change this value.

The Session Store Location field determines where SFSB checkpoint states are persisted. To set the Session Store Location, see To Configure EJB Settings.

Additional Properties

Additional properties for EJB Container Availability.

The GlassFish Server does not define any additional EJB Container Availability properties.
